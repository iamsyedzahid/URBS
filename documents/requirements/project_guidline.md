Project Guidelines
(Deadline: 16th week)
Only 2 or 3 Group Members are allowed.
As a requirement, you will need to submit a course project using the concepts
learned in the theory classes. Following are the few things you should keep in
mind regarding the project.
1. Application of Concepts:
Try to use most of the major concepts of Software Engineering. Make
sure you have all the entities properly connected through relationships
wherever they need to be.
2. Work-Flow Over Jira
You have to show all your work progress throughout your project
development over JIRA.
3. Design a SRS Document
The following content should be considered:
1. Introduction
2. Problem statement
3. Methodology/Workflow
4. Application Architecture
5. Tools & Techniques
6. GUI-Design
7. Functional and no Functional Requirements
8. Testing along with Test Case
9. Any Outsource Libraries
4. Working Project
Create a graphical user interface for your project, applying design principles
learned in this course. Ensure all possible input validations are included.
5. Evaluation Criteria
Your project will be tested in light of your project proposal i.e. all the
features specified in the proposal should be in the finished product.
The project will be of 10 points mapped on the same weightage, which will
be roughly divided as:- Proposal Submission- 1 point- Work-Flow Over Jira-1 point- Project Documentation 2 points //Template shared- Working Project 6 points (including 2 for testing, 1.5 for UI, 1 .5for
architecture, and 1 for Q/A)
6. Plagiarism Policy
Remember, your project will be a key part of your learning in this course. It
won't matter much if you cannot correctly implement a feature since this is
just a learning phase. You are also allowed to check out online references to
remove any errors in your code, but make sure that you don't download or
copy the complete code. Documentation as it will surely be caught during
evaluation and there will be no learning for you that way.
7. Submission
You will be required to submit the code and project report according to the
deadline that will be specified on GCR
